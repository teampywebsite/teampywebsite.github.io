Background

Our client is the Department of General Services. They were working on modernizing their time tracking system. Their current system is not as up to date as they would like it to be. That is where our team comes in. They assigned us the task of designing and implementing a new system.

Scope

The purpose of our project is to reduce the amount time it takes for our client's employees to input their time. There is also a need for our time tracking system to be implement within their own current website.

The business problem this project will address is the current amount of time the employees are spending on the current system.

The main goal of this project is to provide a resource for both employees and supervisors to have a system in place within the current website.

Solution

Our team will improve their tracking system by allowing it to be more efficient with inputting data and allowing all users to access the data tracking system as well as centralizing the process and output

Highlights

Business will now be able to use a more modernized and efficient web application for their new Time Tracking System.

  • Quick and easy for employees to input their task hours
  • Prohibit staff the ability to edit previous times
  • Any time modification can now only be done by supervisors
  • New design will be implemented with using Microsoft .NET Core 3.1.1 and Angular 9.0
